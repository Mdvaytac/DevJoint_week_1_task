# DevJoint_week_1_task
Project Overview

This project focuses on analyzing the Northwind sales database using SQL to extract meaningful business insights and evaluate overall business performance. The analysis covers customer purchasing behavior, product sales, employee performance, category performance, and revenue generation.

The project involves writing intermediate-level SQL queries using multiple table joins, aggregate functions, Common Table Expressions (CTEs), correlated subqueries, and window functions to solve real-world business problems. These techniques are used to transform raw transactional data into valuable information that supports data-driven decision-making.

Key analyses include identifying top customers by revenue, determining best-selling products within each category, evaluating employee sales performance, analyzing customer purchasing diversity across product categories, detecting underperforming products, and ranking employees by revenue within each shipping country.

Additionally, the project emphasizes query optimization by applying best practices such as using indexes, reducing unnecessary calculations, replacing inefficient correlated subqueries with joins when appropriate, and improving query readability with CTEs. These optimizations help enhance query performance and scalability when working with larger datasets.

Overall, this project demonstrates practical SQL skills commonly required for Data Analyst roles, including data retrieval, business analysis, query optimization, and performance reporting. It showcases the ability to convert relational data into actionable insights that can support business strategy and operational improvements.

Project Objectives
- Analyze sales performance using SQL.
- Measure customer, employee, and product performance.
- Generate business insights from transactional data.
- Practice advanced SQL techniques such as CTEs and Window Functions.
- Apply query optimization techniques to improve efficiency.
- Produce analytical reports that support business decision-making.
